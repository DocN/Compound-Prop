Description: This program is used to compound two propositions P and Q, as well as there truth values. 
Compounding Propositions will ask a user for the proposition P and the truth value of proposition P, followed by
the proposition Q and the truth value of proposition Q. Then Compounding Propositions will output the compound
conjunction, disjunction, negation, implication and biconditional propositions and there compound truth values.

----------------------------------------------------------------------------------------------------------------
Usage: 
-Program will ask for a proposition P
*User input proposition P
-Porgram will ask for truth value of proposition P
*User input truth value for proposition P either (TRUE),(T) OR (FALSE),(F)
-Program will output the compound proposition is format of conjunction, disjunction, negation, implication and biconditional propositions.
-Followed by the compound truth values of proposition P and Q in the form of conjunction, disjunction, negation, implication and biconditional truth values.
